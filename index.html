Bem-vindo ao meu site!

Neste site irei fazer uma breve apresentação sobre meus passos na programação.

Como conheci?

Conheci a programação através de aulas de robótica durante o ensino médio. Tive a oportunidade de entender melhor sobre a lógica de programação e ver de perto alguns protótipos de robôs.

Como iniciei os estudos?

Iniciei os estudos em 2022 com alguns vídeos de programação e cursos gratuitos.

O início!

No ano de 2023, iniciei os estudos de Engenharia de Software na UNIALFA, onde curso meu 4º período.

Em que atuo hoje ?

No início de 2024, tive a oportunidade de começar a atuar na área de desenvolvimento de uma empresa que presta serviços para prefeituras. Hoje, tenho mais contato com linguagens de programação como JavaScript e C# usando React como framework pois atuo no Front-end.

Onde quero chegar?

Sempre muito curiosa, quero imergir em cada parte da tecnologia e construir algo que seja importante para um grupo de pessoas. Ainda muito para descobrir sei que nesse caminho posso conseguir ótimas oportunidades.
